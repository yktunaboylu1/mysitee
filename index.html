Classifieds mobile app with backend, REST API and admin panel.

This app is a classified ads based app, where people can buy, sell, rent or giveaway stuff they don't need. users can trade their own products or bid a smaller price on any item they want.

There is an admin panel too, where the super admin can manage anything inside the app (categories, users, products, comments, trades, offers, anything), and there is a REST API service created to handle data back and forth the app and database. As for the database, this is based on MySQL.

The app is powered by Ionic 3 (TypeScript), as for the backend, REST API and admin panel, these are developed with Django (famous library for Python).

To use the app you don't have to know any of these techniques, but only once you decide to change anything inside, or add new features, then yes, you need to be familiar with HTML, CSS, Angular, and for the web app Django / Python, serializers, REST API structure.
